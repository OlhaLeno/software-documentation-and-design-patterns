# software-documentation-and-design-patterns

### Variant 5

Намалюйте use-case діаграму пошуку фільму певної категорії користувачем на платформі.
Намалюйте діаграму класів, щоб представити множину об’єктів необхідних для реалізації збереження, пошуку та представлення інформації про фільми на платформі.
Намалюйте діаграму активності оцінювання кожної серії та додавання відгуків користувачем у платформі.
Намалюйте sequence діаграму, яка показує перегляд відео користувачем на платформі з-допомогою різних пристроїв та яким чином користувач може продовжувати перегляд серіалу або фільму на різних пристроях з моменту призупинення перегляду на іншому пристрої.
